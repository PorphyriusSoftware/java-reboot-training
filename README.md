# Java Reboot — Training Repository

This repository is a structured, incremental training ground for rebuilding and sharpening Java fundamentals.  
It follows a clean, test-driven workflow: each challenge introduces a focused concept, a small implementation, and a dedicated test suite.

The goal is to create a professional, well-documented portfolio of Java skills while reinforcing clean code habits, testing discipline, and modern development practices.
